Here you can lookup the PAThs version of the CMCL data to see if there is a matching entry and if there are Clavis IDs which we can insert in our data.

You have to compare what is on the left (BM) to what is on the right (PAThs/CMCL).

to start searching you can hit "search this" beside one of the existing title and this value will be copied in the input box on the left to start a search, or you can just type into that box, then click out of it and it will do the search in PAThs.

There might be multiple results, which will be all displayed one below the other for you already in an entry form. You can edit those values if you think they are wrong, without touching the PAThs data naturally.

Make sure the data you wish to add is not already there. you see what we already have in the box under the main title of the BM work.

Once you have found a matching work record in PAThs and are happy with the information you are adding, you can hit the Add data to ID button under the form and the file will be update. you will see a confirmation message and you will have to download and commit the file to GIT.

If you find problems you know where to ask.
